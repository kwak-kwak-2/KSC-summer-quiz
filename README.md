# 🎯 공항성산교회 중등부 여름수련회 - 스마트 골든벨 빙고 시스템

> **2026년 공항성산교회 중등부 여름수련회**에서 학생들이 조별로 참여할 수 있도록 구축된 실시간 데이터 연동형 웹 게임 플랫폼입니다. 본 시스템은 진행자 마스터 보드와 학생 대표용 모바일 입력 패널이 Google Firebase를 통해 실시간으로 양방향 동기화됩니다.

---

## 🚀 주요 기능 및 기믹

### 💻 진행자용 마스터 메인보드 (`index.html`)
- **트렌디한 UI/UX**: 다크 네이비 배경과 네 Neon 포인트를 주어 대형 빔프로젝터 화면 투사 시 최적의 시인성을 보장합니다.
- **실시간 조별 상황 모니터링**: 우측 사이드바에서 1조부터 5조까지의 5x5 빙고판 현황과 정답 처리 여부, 현재 실시간 빙고 개수를 마스터 판넬에서 원격 모니터링합니다.
- **정답 스포일러 차단**: 빔프로젝터에 화면이 상시 노출되는 특성을 고려하여, 출제용 참고 정답 레이어를 완벽히 숨겨 진행의 긴장감을 유지합니다.
- **자동 타이머 및 음향 시스템**: 난이도별(최상, 상, 중, 하) 자동 타이머 매핑 및 프로그래스 바 애니메이션이 작동하며, 긴장감 넘치는 효과음(초침 소리, 타임오버 부저, 퀴즈 오픈 짜잔 사운드)이 연동됩니다.

### 📱 학생 조별 대표자용 프로그램 (`mobile.html`)
- **DIY 전략 빙고판 구성**: 조 접속 시 1~25까지의 숫자 칩을 터치하여 우리 조만의 커스텀 5x5 빙고판을 직접 배치하고 서버로 원격 전송합니다.
- **실시간 문제 푸시 팝업**: 선생님이 메인보드에서 문제를 클릭하는 즉시, 해당 번호를 가진 조들의 스마트폰 화면에 **퀴즈 카테고리, 난이도, 퀴즈 본문 내용**이 포함된 단답형 정답 입력창이 강제로 팝업됩니다.
- **실시간 점령 피드백**: 선생님이 채점을 완료하면 본인 조의 5x5 빙고판에 실시간으로 화려한 점령 불빛이 켜지며 사운드 알림이 울립니다.

데이터베이스는 서버 비용이 전혀 발생하지 않는 **Firebase 무료 요금제(Spark)** 범위 내에서 초경량 텍스트 구조로 실시간 동기화됩니다.
